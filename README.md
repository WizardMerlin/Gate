2.2 版本属于优化版本， 走“简”的道路。 更少，更专业。

* 精简，简化： 完全去掉初级内容；太过详细的内容写成小册子。
* 精致，精华： 让精华的，有益处的东西突出。
* 专业化： 博客不再是仅仅为我个人的兴趣爱好服务了，也为我的职业发展服务。
